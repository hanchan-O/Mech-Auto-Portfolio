# Mech-Auto-Portfolio 🚙

> 机械汽车设计作品集 - 从CATIA建模到ANSYS仿真的完整设计流程

## 📋 项目简介

本仓库展示了我从2026年3月到2026年12月的机械设计作品，涵盖CATIA建模、装配设计、ANSYS仿真分析等核心技术。

**目标**: 为秋招提供可展示的机械设计作品集，冲刺车灯、底盘、CATIA技术支持等岗位。

## 🎯 核心项目

### 1. 智能车灯设计项目
**项目路径**: [1-Headlight-Design](./1-Headlight-Design/)

**项目描述**: 特斯拉Model 3前大灯逆向建模与仿真分析

**核心内容**:
- 灯罩复杂曲面建模（CATIA创成式曲面设计）
- 车灯总成装配（灯壳+透镜+支架）
- LED热固耦合分析（ANSYS热分析）
- 光学设计思路探讨（SPEOS入门）

**适配岗位**: 车灯设计、结构设计、光学工程师

---

### 2. 悬架控制臂设计项目
**项目路径**: [2-Suspension-Control-Arm](./2-Suspension-Control-Arm/)

**项目描述**: 麦弗逊悬架控制臂逆向建模与仿真优化

**核心内容**:
- 控制臂1:1逆向建模（球铰、衬套安装位）
- 静力学分析（应力、应变、变形云图）
- 模态分析（固有频率、振型）
- 轻量化优化设计
- DFMEA思考文档

**适配岗位**: 底盘设计、悬架性能件、CATIA技术支持、小鹏悬架培训生

---
